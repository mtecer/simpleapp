# simpleapp
Simple Flask app running in Vagrant demonstrating data processing

# Clone
```shell
git clone https://github.com/mtecer/simpleapp.git
```

# Deployment
```shell
cd simpleapp
vagrant up
vagrant ssh
```

# Testing
```
vagrant ssh
vagrant $ cd /usr/local/python/simpleapp
vagrant $ python python tests/simpleapp-test.py
```
